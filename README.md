# Vietnam Blood Management System - v2.0 (Full FHIR Integration)

This project is the **Phase 2** evolution of the Vietnam Blood Management SOP-10 system, focusing on:
- **Full HL7 FHIR R4 Integration**
- **Alignment with Vietnam National Healthcare Data Center Standards**
- **12-digit CCCD Identification Support**

View your app in AI Studio: https://ai.studio/apps/a67dd908-fefd-4250-855e-35bba1de73a5

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
